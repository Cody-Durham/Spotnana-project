# Spotnana-project
This is a simple web-based app called Curious** that focuses on learning or answering questions.
There are links toward the top of the pabge that allows the user to navigate to vairous sites that promote learning.
The user can also input text into the prompt toward the bottom of the screen and submit to receive data called to OpenAi.
